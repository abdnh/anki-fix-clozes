This Anki add-on tries to fix various issues causing clozes not to function properly. At the moment, the add-on only fixes the case where there are clozes partially enclosed by HTML tags (which will cause Anki to not properly highlight the text in blue). For example, the text hello <span>world {{c1::</span>test}} will be converted to hello <span>world </span>{{c1::test}} Changelog See CHANGELOG.md for a list of changes.
